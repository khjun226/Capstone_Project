## 📌 프로젝트 개요

- **과목**: 2025년 1학기 캡스톤디자인 (4학년 1학기)  
- **프로젝트명**: My Playstify  
- **목적**:  
  1. 노래 검색
  2. 사용자가 검색한 노래에 기반한 노래 추천 기능
  3. 사용자가 직접 플레이리스트 생성  
  4. 플레이리스트에서 직접 노래 재생

## 🚀 주요 기능

1. **노래 검색**  
   - 노래 검색(제목·아티스트)  
   - 검색 결과 썸네일·제목·아티스트 표시  

2. **추천 곡 보기**  
   - 추천 리스트 생성  
   - 앨범 커버·미리 듣기로 확인  

3. **플레이리스트 관리**  
   - 세션에 플레이리스트 저장·불러오기  
   - 추가·삭제·이름 지정  
   - “재생하기” 버튼으로 커스텀 플레이어 이동  

4. **커스텀 플레이어**  
   - 노래 재생 기능  
   - 재생·일시정지·이전·다음·시킹 컨트롤
